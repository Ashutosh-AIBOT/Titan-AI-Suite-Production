## 🚀 Advanced Interview-Ready Features
This version has been significantly enhanced beyond basic learning scripts to include:
- **Rate Limiting**: Built-in protection to prevent API exhaustion.
- **Latency Tracking**: Real-time performance monitoring of LLM response times.
- **Structured Logging**: Professional-grade logging for debugging and observability.
- **Persistent Cognitive Memory**: Advanced SQLite-based session management.
- **Error Resiliency**: Robust handling of tool failures and network issues.

## 🏗️ Architecture
- `advanced_cognitive_agent.py`: A consolidated, high-performance agent featuring memory, tools, and performance tracking.
- `process_orchestrator.py`: Multi-step logic control (Sequential & Parallel).
- `HLD.md`: System design and Mermaid diagrams.

## 🛠️ Setup
1. Create a virtual environment:
   ```bash
   conda activate agent
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set your NVIDIA key in `.env`.
